学会使用高级语言访问 MySQL 数据库,并进行查询。

\begin{itemize}
    \item Ubuntu 16.04.5; MySQL Ver 14.14 Distrib 5.7.25

    \item Java version 1.8.0\_181; IDEA 2018.3.5

\section{实验心得}
本次实验可以选择使用不同的高级语言来完成, 此处用JAVA来完成实验, 相比使用C连接数据库, 仅仅需要将相应的\texttt{jar}包导入即可.
总的来说, 实验的难度不大.
